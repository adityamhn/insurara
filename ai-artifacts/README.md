# AI collaboration artifacts

Raw, unedited `.jsonl` session logs from the coding agents used to build this project —
included so the work can be traced phase by phase (framing, domain research, planning,
coding, docs, testing, QA).

- `claude/` — Claude Code / Claude Desktop sessions (the primary build: engine →
  persistence → API → settlement → disputes → frontend → docs, plus the reviewer MCP).
- `codex/` — Codex CLI sessions.

These are verbatim transcripts as written by each agent — no curation, summarization, or
trimming. They were copied straight from each agent's session directory.
